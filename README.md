# React Admin Dashboard

A modern admin dashboard built using React, Vite, and Tailwind CSS.

## Features

- Dashboard analytics
- Charts with Recharts
- Users management (CRUD)
- Search, sorting, pagination
- Reports filtering and CSV export
- Role-based access control
- Dark / Light theme toggle
- Responsive layout

## Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
- React Router
- React Toastify
